
一、Redis简介
REmote DIctionary Server(Redis) 是一个高性能的key-value数据库。
值（value）可以是 字符串(String), 哈希(Hash), 列表(list), 集合(sets) 和 有序集合(sorted sets)等类型。
(Redis) https://github.com/tporadowski/redis/releases
(Redis GUI) https://github.com/qishibo/AnotherRedisDesktopManager

二、Redis优势
性能极高。读写
丰富的数据类型。
单个操作是原子性的。多个操作也支持事务。
丰富的特性 C Redis还支持 publish/subscribe, 通知, key 过期等等特性。

三、数据类型
	1、string类型
	string 类型是 Redis 最基本的数据类型，string 类型的值最大能存储 512MB。
